# Voron RP2040

Official Windows releases of the Voron RP2040 Configurator.

Voron RP2040 is a configurable keyboard and mouse HID keypad based on RP2040. This repository contains public application releases only. Project source code is not included.

## Download

Open the [Releases](https://github.com/edpol770-spec/Voron-RP2040-Releases/releases) page and download the latest Windows installer:

`Voron-RP2040-Configurator-0.9.0-beta-Setup.exe`

## Installation

1. Download the installer from the Releases page.
2. Run the installer and follow the setup steps.
3. Connect the Voron RP2040 device by USB.
4. Start Voron RP2040 Configurator.

Windows SmartScreen may warn about an unknown publisher because the current beta installer is not digitally signed.
